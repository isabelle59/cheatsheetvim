cheat-sheet VIM

Les bases de VIM


Vim est un éditeur de texte pour le terminal sous GNU/Linux.
Il a une stabilité exemplaire et ne cesse d'être amélioré.
Il est très apprécié des développeurs pour toutes ses fonctions qui en font un très bon IDE
(coloration syntaxique de 200 langages, complétion automatique, comparaison de fichiers, recherche évoluée, …)
et est extensible par des scripts.


1.les commandes pour se déplacer sur le terminal


Ces commandes permettent un mouvement libre.
*k pour naviguer vers le haut
*j pour naviguer vers le bas
*h pour naviguer vers gauche
*l pour naviguer vers la droite


2.Les commandes d'insertions


Ce sont les commandes qui permettent de générer une action.


mode insertion
i=insert passe en mode insertion
a=append passe en mode insertion en faisant avancer le curseur
o=open passe en mode insertion en créant une nouvelle ligne sous le curseur
O=open passe en mode insertion en créant une nouvelle ligne au dessus du curseur
R=replace passe en mode remplacement
mode visuel
v=visual passe en mode visuel
V= passe en mode visuel par ligne
ctrl-V= passe en mode visuel par bloc
mode action
d=delete
qui permet de supprimer c=change y=yank
